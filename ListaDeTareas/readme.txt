Escribe un programa que administre una lista de tareas (to-do list). 
El programa deberá permitir al usuario añadir, ver, actualizar y eliminar tareas. 
Las tareas se guardarán en un archivo de texto llamado tareas.txt, de modo que las tareas se conserven incluso cuando se cierre y vuelva a abrir el programa.

Requisitos
    Manejo de Excepciones: 
        Asegúrate de capturar errores comunes, como el intento de abrir un archivo inexistente o escribir en él.
    Estructura CRUD: 
        Incluye opciones para crear, leer, actualizar y eliminar tareas.
    Funcionalidades:
        Agregar una nueva tarea.
        Ver todas las tareas.
        Actualizar una tarea existente.
        Eliminar una tarea.
    Guardar y Cargar Tareas: 
        Las tareas deben guardarse en el archivo tareas.txt después de cada modificación. 
        Al iniciar el programa, carga las tareas existentes desde el archivo.
    Formato de Archivo: 
        Cada línea en el archivo debe representar una tarea.
